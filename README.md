# Cannibal-Slayer
A retro-inspired 2.5D FPS built in Python using the Ursina engine, featuring 2D environments and 3D sprite entities.
Vision - A Doom-style shooter focusing on destroying cannbibals on a certain island for learning game development.
Tech Stack - Python 3.10 and the Ursina engine.
Installation instructions: pip install ursina and run python main.py
Controls - WASD to move, Space to jump, Mouse to shoot
